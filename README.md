# EcomLocalization-LLM
# 基于LLM实现跨境电商的本土化

## 使用前准备
1、需要先安装chrome浏览器以及chromedriver  

2、下载DrissionPage包，  
pip install DrissionPage

3、把你的目标商品的ID,评论保存地址，最大爬取页数更换即可。（注：京东网页版最多可加载19页评论）   商品详情页面网址的一串数字就是商品ID。
